# Generador y Clasificador de juegos
## este proyecto consisten en probar algoritmos de ordenamiento y lectura sobre archivos .csv, en este caso existen 3 opciones posibles:
### 100
### 1000
### 10000
## Como usar:
### Primero es necesario tener un compilador de java, para ello visitar la pagina oficial de Oracle
### Luego ejecutar lo siguiente:
#### 'java GenerateData.java' y luego 'java Main.java'
#### Elegir las opciones disponibles en la terminal
